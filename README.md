# water-generator

Agua: Con el vapor de agua que hay en el aire obteniéndolo y condensándolo.
Agua2: Juntando 2 atomos de hidrogeno y 1 de oxigeno a partir de una descarga electrica,esto podria ser altamente peligroso debido a su inflamabilidad.
